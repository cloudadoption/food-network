Simple Template Test Page

This page demonstrates the simple template with basic styling and minimal JavaScript functionality.

Features

The simple template provides:

  • Clean, centered layout
  • Consistent color scheme
  • Styled links and headings
  • Minimal JavaScript enhancements

Template Loading

This template demonstrates the three-phase loading system:

  1. Eager phase: Adds a data attribute to track template loading
  2. Lazy phase: Adds a simple banner to indicate the template is active
  3. Delayed phase: Logs completion (minimal functionality)
